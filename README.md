# README
開発言語
Ruby 3.0.1
Rails 6.1.7
PostgreSQL 12.8

実行手順
$ git clone https://github.com/aikahedwig/recipe_app.git
$ cd recipe_app
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
